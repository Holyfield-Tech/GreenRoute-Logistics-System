# GreenRoute-Logistics-System
An innovative C++ system for tracking carbon footprints in logistics using OOP principles.

## 📖 Overview
GreenRoute is a C++ system designed to help logistics companies track and reduce their carbon footprint. It uses Object-Oriented Programming (OOP) to simulate a fleet of different vehicle types and calculate environmental impact based on distance and load.

## ✨ Key Features
- **Carbon Tracking:** Differentiates between EV and Bio-fuel impacts.
- **Route Optimization:** Uses a 2D distance matrix for hub-to-hub calculations.
- **Cargo Merging:** Custom operator overloading allows for seamless shipment consolidation.
- **Error Handling:** Robust custom exception classes to prevent system crashes.

## 🛠️ OOP Principles Applied
- **Inheritance:** Single, Multilevel, and Multiple Inheritance models.
- **Polymorphism:** Dynamic binding used to process a diverse vehicle fleet.
- **Encapsulation:** Secure data management with private members and public interfaces.

## 🚀 How to Run
1. Clone the repository
2. Compile the code: `g++ main.cpp -o GreenRoute`
3. Run the application: `./GreenRoute` (Linux/Mac) or `GreenRoute.exe` (Windows)
